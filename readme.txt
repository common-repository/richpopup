RichPopUp is a plugin which displays with javascript popup showing 6 links to your recent posts. It shows when person visiting our blog is leaving the site. 
This popup will be shown only once on the same page. 

== Description ==
RichPopUp is a plugin which displays with javascript popup showing 6 links to your recent posts. It shows when person visiting our blog is leaving the site. 
This popup will be shown only once on the same page. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
